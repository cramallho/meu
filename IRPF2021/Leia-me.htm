LEIA-ME

IRPF 2021

Este texto oferece ao usuário do programa IRPF 2021 algumas informações úteis sobre sua utilização, bem como instruções para resolver eventuais problemas de incompatibilidade na instalação.

Detalhes para uso encontram-se no Ajuda do programa.

1. REQUISITOS E INSTALAÇÃO

1.1 Requisitos necessários para utilizar o programa:

  1. Para os sistemas operacionais LINUX, WINDOWS e MAC, a Máquina Virtual Java está embutida na aplicação e será instalada juntamente com ela.

  2. Navegador de Internet (Browser)

  3. Em caso de sistema operacional Windows, a versão mínima recomendada é o Windows 7 ou superior

1.2 Instalação do programa:

O programa pode ser obtido na página da Secretaria Especial da Receita Federal do Brasil ou nas unidades da RFB. Se não conseguir executar o instalador, consulte o administrador do seu sistema para averiguar se não é necessário alterar permissões de usuário ou quota de disco.

2. CONFIGURAÇÃO DO COMPUTADOR

2.1 Resolução de vídeo e tamanho de fontes

Para utilizar o programa IRPF 2021, é necessário que o monitor esteja configurado para 1280 x 768 pixels ou maior e fontes pequenas.

Caso esteja utilizando o sistema operacional Windows, para alterar a resolução da tela (recomendável), clique no menu Iniciar do Windows, selecione Configurações>Painel de Controle>Vídeo. Clique na guia Configurações e selecione a área de trabalho para 1280 por 768 pixels ou superiores para fontes pequenas.

Para outros sistemas operacionais ou gerenciadores de janela, consulte o respectivo manual do usuário.

4. SOLUÇÃO DE OUTROS PROBLEMAS

Usuários de Beryl/Compiz

Em algumas instalações Linux utilizando o Beryl (http://www.beryl-project.org/) as janelas do IRPF 2021 podem aparecer sem conteúdo, sendo exibido apenas o contorno das mesmas. Para sanar este problema, antes de carregar o IRPF 2021, utilize o seguinte comando:

export AWT_TOOLKIT=MToolkit
